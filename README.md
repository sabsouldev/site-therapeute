# 🌿 Site vitrine fictif - Claire Dupin, Thérapeute holistique

Ce projet est un **site one-page responsive** fictif, conçu pour une thérapeute à Bordeaux, dans le cadre de mon portfolio professionnel.

## ✨ Objectifs
- Créer une page d’accueil claire et accueillante
- Présenter les services de façon lisible (cartes Bootstrap)
- Proposer un formulaire de contact simple
- Utiliser Bootstrap 5 pour un rendu responsive rapide

## 🔧 Tech utilisées
- HTML5
- CSS3 (fichier séparé)
- Bootstrap 5 (CDN)
- JavaScript de base (fichier vide prêt à l’emploi)
